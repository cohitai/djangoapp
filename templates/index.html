Here it will be described what the project is, and what was done so far.

Outline: What we do is structure-from-motion (sfm) with the open source COLMAP for 3D-reconstruction of plants.

The generated point clouds are stored in a cloud database and analyzed with Open3D for features extraction.

The experiment setup:

We use 2 Raspberry Pi HQ high quality cameras connected to a Raspberry Pi controller for taking videos of the plants from 2 angles. The vidoes are taken live and then streamed into an image database on Azure Cloud.
